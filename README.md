📊 AI Prompt Engineering for Retail Analytics (RCTTCE Framework)

📌 Project Overview
This project applies the RCTTCE (Role, Context, Task, Format, Constraints, Example) framework to design an AI prompt for analyzing business performance in a retail environment.
The case study focuses on a retail company operating 30 stores that has experienced declining performance over the last 18 months. The goal is to support the CEO and analytics team in identifying causes of decline and opportunities for improvement.

🧠 Problem Context
The retail company provided a dataset of approximately 50,000 records covering:
Store performance
Product sales
Customer transactions
Loyalty programme activity
The objective is to analyze these data areas to understand performance trends and customer behavior patterns.
🎯 Project Objectives
Design a structured AI prompt using the RCTTCE framework
Generate SMART business questions to guide analysis
Translate findings into a stakeholder-friendly business brief
Support data-driven decision-making for retail improvement
